# Student Management System

A Spring Boot based Student Management System developed for managing student records.

## Features
- Add Student
- View Student
- Edit Student
- Delete Student
- Search Student

## Technologies Used
- Java
- Spring Boot
- Thymeleaf
- H2 Database
- HTML/CSS
